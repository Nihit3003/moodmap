# Smart Nearby Places Recommender

A web application that recommends nearby places based on the userâ€™s **mood and intent**, helping users quickly find the most suitable spots around them.

# What does it do?

- User selects or enters a **mood** such as:
  - Work
  - Date
  - Quick bite
  - Budget-friendly
- The app fetches **nearby places** relevant to that mood
- Displays useful details including:
  -  Distance
  -  Ratings
  -  Open hours
- Results can be **filtered and sorted** for better decision-making

---

# Tech Stack

- **Frontend:** JavaScript, React
- **APIs Used:**
  - Google Maps API
  - Google Places API
- **Core Features:**
  - Mood-based recommendation logic
  - Filters & sorting
  - Location-based search
